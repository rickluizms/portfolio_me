Dashboard Power BI - Desafio de Projeto DIO

Este painel utiliza dados de amostra da internet para criar um relatório e análise de dados dinâmica. Com gráficos interativos e insights valiosos, o projeto demonstra a capacidade de utilizar o Power BI para transformar dados em informações acionáveis.

Micro Sistema VBA - McSoft

O McSoft é um micro sistema baseado no Excel com VBA. Ele simplifica o gerenciamento financeiro, oferece análises de dados avançadas e facilita a tomada de decisões informadas para pequenas empresas.

Web App - Didata Docs

Web App - Didata Docs é o Front-End que se integra ao Didata Docs Service, proporcionando uma interface amigável para os usuários acessarem e utilizarem o serviço de geração de documentos sem a necessidade de desenvolver um cliente para a API. Desenvolvido com Python no framework Django, esse aplicativo torna a criação de documentos simples e acessível.

API Services - Didata Docs Service

API Services - Didata Docs Service é uma API versátil e personalizada que gera documentos, como certificados, cartas e termos de responsabilidade, de acordo com as solicitações do usuário. Integrada ao banco de dados da empresa, especialmente escolas, esta solução garante a criação eficiente e precisa de documentos para atender às necessidades da organização contratante.

API Services - UserNine Cookies Service

API Services - Didata Cookies Service é uma API REST robusta desenvolvida em Java 17 e Spring Boot. Com recursos de CRUD, DTO e segurança aprimorada com tokens JWT, essa solução oferece um serviço confiável e escalável para gerenciar dados de cookies de forma eficiente.

API Services - UserNine Disc Service

API Services - Didata Disc Service é uma API altamente funcional que se destina à geração e correção de formulários de perfil DISC, fornecendo resultados precisos de testes de personalidade. Amplamente adotado pelo setor de Recursos Humanos, este serviço simplifica a análise de perfis de funcionários, facilitando a identificação de características de personalidade.

Web Scraping - Didata Data Scraping

Serviço web desenvolvido com Python 3.9 e Selenium para atualizar o banco de dados com os dados raspados na plataforma da empresa. O bot automatiza processos repetitivos e extrai informações uteis para planilhas, autmentando a produtividade da equipe.
